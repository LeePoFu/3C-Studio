# 3C Studio v5.3

## 雙人對戰升級
- 新增「目標分數」與「限時驟死賽」兩種模式。
- 目標分數與限時驟死賽皆可設定「有後攻／無後攻」。
- 有後攻時：白球達標後可繼續本杆；失誤換黃球最後後攻；黃球超越即勝、落後則白球勝、追平進入延長賽。
- 限時驟死賽顯示倒數；無後攻時時間到直接判定；有後攻時讓當前球員完成本杆，並確保黃球取得最後後攻，追平則進入驟死延長。
- 修正結束比賽摘要無法顯示的 JavaScript 重複宣告錯誤。
- 剩餘分數提示改為正常版面流，不再覆蓋計分按鈕。

## v5.3.1 更新
- 限時模式保留後攻選項。
- 時間到時，白球若正在出杆可完成本杆，之後黃球取得最後一次後攻。
- 時間到時若黃球正在出杆，該杆即視為最後後攻。
- 後攻結束平手時進入驟死延長；否則依分數判定勝負。
