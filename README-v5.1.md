# 3C Studio v5.1

## 活動架構
首頁改為三大活動入口：
1. 個人練習
2. 雙人對戰
3. 多人追分

一般對戰與正式比賽已整併至同一套比賽計分板。
活動／比賽名稱與賭注改為選填欄位：
- 活動名稱空白：一般對戰
- 活動名稱有填：可作為正式比賽、球聚或其他活動名稱
- 賭注空白：無賭注
- 賭注有填：保留於該場活動資料

## 比賽計分
- 預設目標分數改為 20 分
- 新增日期、時間、地點、活動名稱與賭注欄位
- 結束比賽後先顯示摘要
- 使用者選擇「保留紀錄」後才寫入歷史
- 選擇「不保留」則捨棄該場資料

## UI
- 比賽設定頁改為與 Records 相同的淺色、白色卡片及藍綠主色
- 統一按鈕、圓角、卡片、陰影與結果對話框
- 比賽進行畫面保留高辨識度，但改用同一套 3C Studio 色彩

## 部署
將 ZIP 內容全部覆蓋至目前的 3C Studio / 3C-records Repository 根目錄後 Commit、Push。
iPhone 或 iPad 若仍顯示舊版，請關閉主畫面 App 後重新開啟；必要時刪除舊圖示並重新加入主畫面。
